# Developer mode

1. Clone repository
2. Exec:
```
yarn install
```
3. Installed Nest CLI
```
npm i -g @nestjs/cli
```
4. Start database (docker)
```
docker-compose up -d
```

## Technologies
* Nest
* Node
* MongoDB
* Docker
* HTML + CSS
